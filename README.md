# Python practice

This repository contains small Python exercises and experiments created while learning core Python concepts.

## Topics covered
- Dictionaries and basic data structures
- File input/output
- Matrix processing
- Simple command parsing

## Structure
- `dictionary/` — exercises with Python dictionaries
- `files/` — file read/write tasks
- `matrix/` — matrix input and processing
- `parcing_command/` — simple command parsing logic

## Purpose
This repository is educational.
It is intended to demonstrate basic Python proficiency,
clean code structure, and familiarity with core language features.

## Language
- Python 3
